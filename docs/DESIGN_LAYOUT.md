# Toolsail 前台页面布局设计
## Design Layout & Wireframes for Public Frontend

---

## 0. 通用布局结构

### 0.1 全局导航栏（Header）

**位置**：页面顶部，固定粘性
**高度**：64px
**内容布局**：

```
┌──────────────────────────────────────────────────────────────────┐
│  [Logo]  [Home] [AI Tools] [Digital Tools] [Blog]    [🔍] [🌙] [👤] │
└──────────────────────────────────────────────────────────────────┘
```

**组件分布**：
- 左侧：Logo（24x24px）+ 网站名称 "Toolsail"
- 中间：导航链接（仅桌面端，md 以上）
  - Home
  - AI Tools
  - Digital Tools
  - Blog
- 右侧：
  - 搜索按钮（放大镜图标）
  - 主题切换（太阳/月亮图标）
  - 语言切换（地球图标，可选）
  - 用户菜单（头像/登录按钮）
- 移动端（< md）：汉堡菜单 ☰ 代替导航链接

### 0.2 页脚（Footer）

**位置**：页面底部
**背景**：深灰色（Light: #F5F5F5，Dark: #1A1A1A）
**内容布局**：

```
┌───────────────────────────────────────────────────────────────────┐
│                                                                     │
│  [关于] [隐私政策] [服务条款] [联系我们]  |  [推特] [GitHub]       │
│                                                                     │
│  © 2025 Toolsail. All rights reserved. | contact@toolsail.top     │
│                                                                     │
└───────────────────────────────────────────────────────────────────┘
```

**布局**：
- 三列分布（桌面端）
- 单列分布（移动端）
- 内边距：48px（桌面）/ 24px（移动）
