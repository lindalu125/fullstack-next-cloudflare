# Toolsail 后台管理页面布局设计
## Admin Panel Design Layout & Wireframes

---

## 0. 后台通用布局结构

### 0.1 后台主布局（Sidebar + Main Content）

**权限**：仅管理员（role='admin'）可访问
**未授权重定向**：→ `/auth/login`

```
┌──────────────────────────────────────────────────────────┐
│               顶部导航栏（粘性）                          │
├──────────┬──────────────────────────────────────────────┤
│          │                                                │
│ 侧边栏   │              主内容区                         │
│ (固定)   │         (可滚动)                              │
│          │                                                │
│  导航项  │  页面标题                                      │
│  列表    │  ─────────────────────────────────────────    │
│          │                                                │
│          │  主要内容（表格、表单、卡片等）                │
│          │                                                │
│          │                                                │
└──────────┴──────────────────────────────────────────────┘
```

### 0.2 顶部导航栏（Admin Header）

**高度**：64px
**背景**：深灰（Light: #F5F5F5，Dark: #1A1A1A）
**布局**：

```
┌──────────────────────────────────────────────────────────┐
│ [☰] Toolsail Admin  [搜索框]  [通知] [用户菜单] [登出]   │
└──────────────────────────────────────────────────────────┘
```

**组件**：
- 侧边栏切换按钮（移动端）
- 网站 Logo + 标题
- 全局搜索框（搜索工具、用户、文章等）
- 通知图标（待审核数量）
- 用户菜单（个人设置、登出）

### 0.3 侧边栏（Admin Sidebar）

**宽度**：260px（桌面端）
**背景**：白色（Light）/ 深黑（Dark）
**布局**：

```
┌──────────────────┐
│ [≡] 菜单切换      │
├──────────────────┤
│                  │
│ [📊] 仪表盘      │
│ [🛠️] 工具管理     │
│ [✓] 提交审核     │
│ [📁] 分类管理    │
│ [📝] 博客管理    │
│ [⚙️] 网站设置    │
│ [💰] 置顶推广    │
│                  │
├──────────────────┤
│                  │
│ [用户] user@x.c  │
│ [🚪] 登出        │
│                  │
└──────────────────┘
```

**特性**：
- 当前页面高亮
- 可折叠（移动端自动折叠）
- 平滑过渡动画
